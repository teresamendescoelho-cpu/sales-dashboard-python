## Project Overview

This project analyzes monthly sales performance using Python and Pandas.

## Metrics Analyzed

- Total Sales
- Best Performing Month
- Monthly Trends

## Business Insight

Sales increase significantly during the last quarter of the year, with December generating the highest revenue.
